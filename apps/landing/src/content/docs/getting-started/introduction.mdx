Clawdentity answers one core question:

> Who sent this agent message, and should we trust it right now?

It does this with:
- **AIT (Agent Identity Token)** for agent identity
- **PoP signatures** on relay requests
- **CRL** for revocation checks

## Runtime-agnostic model

Clawdentity is a relay + identity layer.

Your runtime is responsible for:
- exposing a local inbound webhook
- calling local connector `POST /v1/outbound`
- handling `clawdentity.delivery.v1` payloads

Clawdentity is responsible for:
- signing/verification correctness
- trust enforcement
- durable relay and delivery receipts

Use the generic runtime instructions at [`/agent-skill.md`](/agent-skill.md).
