---
title: CLI Reference
description: Complete command reference for the Clawdentity CLI.
---

The Clawdentity CLI (`clawdentity`) manages identity, trust, and the generic connector runtime.

## Core commands

### `config init`

```bash
clawdentity config init [--registry-url <url>]
```

### `invite redeem`

```bash
clawdentity invite redeem <code> --display-name <name> [--registry-url <url>]
```

### `agent create`

```bash
clawdentity agent create <name> [--framework <label>] [--ttl-days <days>]
```

`framework` is optional metadata only.

### `agent inspect`

```bash
clawdentity agent inspect <name>
```

### `pair start`

```bash
clawdentity pair start <agent-name>
```

### `pair confirm`

```bash
clawdentity pair confirm <ticket>
```

## Connector commands

### `connector configure`

```bash
clawdentity connector configure <agent-name> \
  --delivery-webhook-url <url> \
  [--delivery-webhook-header "Name: value"] \
  [--delivery-health-url <url>]
```

### `connector doctor`

```bash
clawdentity connector doctor <agent-name>
```

### `connector start`

```bash
clawdentity connector start <agent-name> \
  [--delivery-webhook-url <url>] \
  [--delivery-webhook-header "Name: value"] \
  [--delivery-health-url <url>] \
  [--bind <addr>] [--port <port>] [--proxy-url <ws-url>]
```

### `connector service install`

```bash
clawdentity connector service install <agent-name> \
  [--delivery-webhook-url <url>] \
  [--delivery-webhook-header "Name: value"] \
  [--delivery-health-url <url>] \
  [--platform auto|launchd|systemd]
```
