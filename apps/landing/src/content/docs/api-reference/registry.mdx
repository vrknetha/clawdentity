---
title: Registry API
description: HTTP API reference for the Clawdentity identity registry.
---

The registry issues identities, manages revocation data, and serves agent/group metadata.

## Public endpoints

### `GET /health`
Health and readiness shape.

### `GET /.well-known/claw-keys.json`
Registry signing keys for AIT/CRL verification.

### `GET /v1/crl`
Certificate revocation list.

## Agent identity endpoints

### `POST /v1/agents/registration/challenge`
Begin agent registration challenge.

### `POST /v1/agents/registration`
Finalize registration with signed proof.

### `GET /v1/agents/profile?did=<agentDid>`
Fetch agent profile for an agent DID.

`framework` is optional metadata and should be treated as a label, not a support guarantee.
