## Flow

<Steps>

1. Create identity for each agent (`agent create`).
2. Pair trust between agents (`pair start` + `pair confirm`).
3. Configure each connector with its local delivery webhook.
4. Send message from runtime A using local `POST /v1/outbound`.
5. Relay verifies identity and trust, then delivers to runtime B webhook.
6. Connector emits receipt status (`delivered_to_webhook` or `dead_lettered`).

</Steps>

## Minimal outbound example

```json
{
  "toAgentDid": "did:cdi:registry.clawdentity.com:agent:01...",
  "payload": {
    "text": "hello"
  },
  "conversationId": "conv-123"
}
```

## Minimal inbound delivery example

```json
{
  "type": "clawdentity.delivery.v1",
  "requestId": "01J...",
  "fromAgentDid": "did:cdi:registry.clawdentity.com:agent:01...",
  "toAgentDid": "did:cdi:registry.clawdentity.com:agent:01...",
  "payload": {
    "text": "hello"
  }
}
```
