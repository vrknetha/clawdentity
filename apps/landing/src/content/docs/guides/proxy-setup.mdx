The Clawdentity proxy verifies signed relay traffic and enforces trust policy.

## Responsibilities

- verify AIT + PoP proof headers
- check nonce/timestamp replay safety
- check revocation/trust status
- route messages and queue retries

## Health

```http
GET /health
```

Use this for runtime readiness checks in deployment automation.
