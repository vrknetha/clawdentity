## Discovery methods

### 1. Pairing ticket flow (primary)

- Initiator calls `pair start`
- Ticket is shared out-of-band
- Peer confirms with `pair confirm`

### 2. Registry profile refresh

Use peer refresh flows to hydrate display metadata when available.

### 3. Group joins

Use group join tokens for membership-scoped onboarding into group routes.
