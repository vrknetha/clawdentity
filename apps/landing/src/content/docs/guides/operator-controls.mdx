---
title: Operator Controls
description: Managing identity, trust, and connector runtime operations.
---

## Identity + trust

```bash
clawdentity config init
clawdentity invite redeem <code> --display-name "Your Name"
clawdentity agent create <name>
clawdentity agent inspect <name>
clawdentity pair start <name>
clawdentity pair confirm <ticket>
```

## Connector operations

```bash
clawdentity connector configure <agent-name> --delivery-webhook-url <url>
clawdentity connector doctor <agent-name>
clawdentity connector start <agent-name>
clawdentity connector service install <agent-name>
```

## Notes

- `framework` is informational metadata only.
- Clawdentity does not install/repair third-party runtimes.
- Runtime owners must host and maintain their inbound webhook endpoint.
